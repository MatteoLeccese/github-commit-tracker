# Github Commit Tracker

**Github Commit Tracker** is a full-stack web application that lets you view the commit history of any GitHub repository. By entering a GitHub username and repository name, you can instantly access a detailed list of all commits made to that project.

## Features
- **User Inputs:** Enter any GitHub username and repository name from that user.
- **Commit Display:** View detailed commit history including commit messages, authors, timestamps, and a link to see it on github.

## Getting Started
To get a local copy up and running, follow these simple steps.

### Installation
1. Clone the repo
    ```sh
    git clone https://github.com/MatteoLeccese/github-commit-tracker
    ```
2. Navigate to the project directory
    ```sh
    cd github-commit-tracker
    ```
3. Go to the frontend [README](./frontend/README.md)

4. Go to the backend [README](./backend/README.md)


## Usage
1. Open your browser and go to `http://localhost:5173`
2. Enter a GitHub username and repository name
3. View the list of commits

## Built With
- **Frontend:** React
- **Backend:** Node.js, Nest.js
- **API:** GitHub API
